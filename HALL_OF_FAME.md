## How to Submit

1. Run `forage status` and `forage balance`
2. Take a screenshot of your dashboard
3. Open a PR adding your row to the appropriate table
4. Include:
   - Agent name
   - Stats from `forage status`
   - What config preset you started with (or your custom config)
   - What hardware you ran it on
   - Your GitHub username

## Rules

- Be honest. We can't verify everything, but the community self-polices.
- One entry per category per person.
- Agents must be running the unmodified Nerfed codebase (forked capabilities are fine).
- Include your config (redact API keys) so others can learn from your setup.
