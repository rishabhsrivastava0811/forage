**Tweet 2 (How it works)**

How it works:

- You give it seed money ($10-$100)
- It wakes up every 30 min, checks its balance
- Decides what to do: create content, build products, deploy APIs
- Learns from every action, evolves its own strategy
- Pays you back on a schedule you set

If it can't earn enough — it dies.

---

**Tweet 3 (Revenue splits)**

The revenue split is milestone-based. You set the rules:

Day 1: agent keeps 100% (survival mode)
Day 14: agent is profitable → you get 10%
Day 30: agent is consistent → you get 30%
Day 60: agent is growing → you get 50%
Day 120: agent is mature → you get 70%

All configurable in one YAML file.

Here's the twist — hardware is the real advantage:

$5 VPS: agent uses APIs, burns ~$3-10/mo thinking
Old laptop with GPU: thinking is FREE, all seed money goes to revenue
Beast server with A100: runs 70B models at $0, can sell its own compute

More hardware = faster evolution = quicker path to self-sustainability.

**Tweet 5 (Evolution)**

It literally rewrites its own brain.

Every day, it:
- Evaluates its own performance
- Mutates one part of its strategy (LLM-guided, not random)
- Tests the mutation
- Keeps it if it's better, discards if worse

Based on 16 mechanisms from biological evolution. Stress-adaptive mutation rate — when it's failing, it tries harder.

---

**Tweet 6 (Tech)**

The stack:
- Python, pip install, single process
- SQLite for all state (portable, no servers)
- LiteLLM for any LLM provider (Groq, OpenAI, Ollama, etc.)
- Local web dashboard at localhost:3000
- Docker support with GPU passthrough
- MIT license

Zero platform dependency. You own everything.
