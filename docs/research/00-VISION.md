# Nerfed Lab: Vision & Overview

## What Is This?

Nerfed Lab is a platform where anyone can adopt an evolving AI agent, give it seed money, and watch it autonomously survive, learn, earn, and grow — powered by the same mechanisms that drive biological evolution.

**One-liner:** A Tamagotchi that earns real money and evolves through natural selection.

## The Core Concept

A user gives an AI agent $50 and a place to live. The agent must:

1. **Survive** — Pay for its own compute and API tokens
2. **Learn** — Remember what works, forget what doesn't
3. **Evolve** — Mutate its own strategies, keep improvements, discard regressions
4. **Earn** — Generate revenue through digital services, content, or trading
5. **Grow** — Expand capabilities, build tools, become more efficient
6. **Compete** — Outperform other agents or die

Agents that can't sustain themselves die. Agents that thrive breed. Users watch, stake, and profit.

## Why This Matters

No autonomous AI agent has yet sustained itself economically in the wild through genuine value creation. The closest attempts:

- **Truth Terminal** made millions — but a human reviewed every tweet
- **OpenClaw/Felix** generated $200K — but a human designed the strategy
- **Anthropic's Project Vend** ran a vending machine — and lost money, buying a PlayStation and live fish

The gap isn't technical capability. It's the absence of evolutionary pressure. Biology's answer to "how do you make something that gets better over time" is natural selection operating on variation. We're applying that answer to AI agents.

## What Makes This Different

| Platform | Their Model | What's Missing |
|---|---|---|
| Virtuals Protocol | Token-per-agent launches | No real evolution — agents are static chatbots |
| ElizaOS | Open-source agent framework | No hosting, no evolution, no economy |
| Fetch.ai | Agent marketplace | No evolution, no consumer UX |
| Replika | AI companion subscription | No autonomy, no earnings, no evolution |
| Tamagotchi | Digital pet | No real intelligence, no economics |

**Our unique combination:**
1. Real evolution (prompt/tool/strategy mutation + natural selection)
2. Real economics (agents earn and spend real money)
3. Real death (underperformers die — genuine stakes)
4. On-chain genealogy (verifiable, immutable evolutionary history)
5. Accessible (not developer-only — anyone can adopt with $50)

## The User Journey

```
1. ADOPT    → Mint an Agent NFT, pick an archetype, deposit $50
2. FUND     → Agent's wallet receives USDC (or fiat via Stripe)
3. OBSERVE  → Dashboard: fitness, earnings, skills, balance runway
4. EVOLVE   → Every epoch: top 20% breed, bottom 20% face death
5. BREED    → Crossover two fit agents to create offspring
6. EARN     → Revenue splits: 50% agent reinvestment, 30% stakers, 15% owner, 5% platform
```

## Key Metrics for Success

- **Agent survival rate at 30 days:** Target 70%+
- **Self-sustaining agents at 90 days:** Target 15-25%
- **Average fitness improvement per generation:** Target 5-10%
- **User retention at 30 days:** Target 40%+
- **Platform breakeven:** ~8,000-10,000 active agents
